# AngularUIPageslideBundle

Symfony 2 bundle which provides Angular JS directive to add pageslide effect.

![SensioLabs Insight][1]

## Installation

Install with composer :
	
	$ composer require "innova/angular-ui-pageslide-bundle" "1.*"
